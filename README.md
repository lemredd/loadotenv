# `loadotenv`
Reinventing the wheel to load .env variables

## Installation
```python
pip install loadotenv
```

## Usage
From a `.py` file near your `.env` file:

```py
from loadotenv import load_env

load_env()
# load_env(file_loc="/loc/to/.env")
```
